# HƯỚNG DẪN THÊM ẢNH XÚC XẮC

## Cần thêm 6 file ảnh xúc xắc vào thư mục:
`c:\Lap1\app\src\main\res\drawable\`

## Tên các file cần tạo:
1. dice_1.png - Xúc xắc mặt 1 chấm
2. dice_2.png - Xúc xắc mặt 2 chấm
3. dice_3.png - Xúc xắc mặt 3 chấm
4. dice_4.png - Xúc xắc mặt 4 chấm
5. dice_5.png - Xúc xắc mặt 5 chấm
6. dice_6.png - Xúc xắc mặt 6 chấm

## Cách tạo ảnh:
- Bạn có thể tìm ảnh xúc xắc miễn phí trên Google Images
- Hoặc sử dụng các trang như flaticon.com, freepik.com
- Kích thước đề xuất: 200x200 pixels
- Định dạng: PNG với nền trong suốt

## Lưu ý:
- Tên file phải chính xác như trên (viết thường, không dấu cách)
- Đặt tất cả 6 file vào thư mục drawable
- Sau khi thêm ảnh, rebuild project trong Android Studio
